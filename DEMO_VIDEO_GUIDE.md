# Demo Video Recording Guide

## Overview
This guide provides a complete script for recording a 3-5 minute demo video showcasing the platform's key features for investor presentations.

## Pre-Recording Checklist

### Technical Setup
- [ ] Screen recording software ready (Loom, OBS, or QuickTime)
- [ ] Recording resolution: 1920x1080 (Full HD)
- [ ] Frame rate: 30-60 fps
- [ ] Audio microphone tested (if doing voiceover)
- [ ] Browser zoom at 100%
- [ ] Close unnecessary tabs and applications
- [ ] Disable notifications (Do Not Disturb mode)
- [ ] Clear browser history/cookies for clean session

### Demo Environment
- [ ] All demo accounts working and tested
- [ ] Database seeded with demo data
- [ ] Internet connection stable
- [ ] Staging/production URL ready
- [ ] Test run completed successfully

### Demo Data Verification
```
Login to each account and verify:
- Client Demo: Has 2-3 active jobs
- Professional Demo: Has quotes, messages, earnings
- Admin Demo: Dashboard shows statistics
```

## Recording Strategy

### Option A: Single Continuous Flow (Recommended)
Record one 4-minute video showing complete user journey
- **Pros**: Natural flow, shows real user experience
- **Cons**: Must be perfect, no editing

### Option B: Segmented Recordings
Record 6-8 separate clips, edit together
- **Pros**: Easier to perfect each section
- **Cons**: More post-production work

**Recommendation**: Option A for authenticity, Option B if multiple takes needed

## üé§ Voiceover Recording Tips

### Setup
- Quiet room, no echo
- USB microphone > built-in laptop mic
- Record in 16-bit, 44.1kHz WAV format
- Use pop filter or sock over mic

### Delivery
- Speak clearly, moderate pace (not too fast)
- Smile while speaking (improves tone)
- Emphasize key words: "AI-powered", "secure", "real-time"
- Pause briefly between sentences
- Record in one take per section (easier to sync)

### Editing
- Normalize audio levels (-3dB peak)
- Remove mouth clicks and breaths
- Add subtle background music (royalty-free)
- Fade in/out at transitions

## üé® Post-Production Enhancements

### Essential
- **Title Card**: "CS Ibiza Platform Demo" (3 seconds)
- **Lower Thirds**: Label each section (e.g., "Client Dashboard", "Professional Matching")
- **Zoom Effects**: Highlight specific UI elements (2x zoom on important features)
- **Cursor Highlighting**: Make cursor more visible (yellow ring)
- **Transitions**: Smooth fades between scenes (0.5s)

### Advanced
- **Annotations**: Arrows pointing to key features
- **Split Screen**: Show client + professional views simultaneously
- **Speed Ramping**: Slow-mo on important clicks, fast-forward on forms
- **B-Roll**: Insert logo, team photos, or office shots
- **Subtitles**: Add captions for accessibility and silent viewing
