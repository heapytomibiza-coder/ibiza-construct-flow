## üí° Pro Tips for Demo Success

### For Screenshots
- Take screenshots at the same time of day (consistent lighting/time stamps)
- Use incognito mode to avoid extensions showing
- Zoom browser to exactly 100%
- Capture 2-3 of each shot (backup copies)

### For Video
- Smile while doing voiceover (improves tone)
- Slow down mouse movements (easier to follow)
- Pause 1-2 seconds on key screens
- Record in quiet room (close windows, turn off AC)

### For Presentation
- Tell a story, don't just show features
- Start with the problem, then show the solution
- Use "we" language (inclusive, builds connection)
- End with clear call to action
